# OOP---Project-3--Password-Application
Exception handling with password verification

Elia Schwartz, COP3330C-33777, May 23,2025

Purpose: The purpose of this code is to learn how to incorporate exceptions in a program to
prevent errors using try-catch blocks. The assignment simulates how we have company employees create
their usernames and passwords by following specific requirements, and determining any errors by displaying 
a message to guide them to enter the right one.

The user inputs are unique usernames and passwords that obey the rules.
The Output will display that both are accepted and created for each personnel.

There is a separate application/file that provides a simple arithmetic exception handling when the user tries to multiply a number by zero.
